About this app

Language is a fundamental aspect of cultural identity, and the preservation of lesser-known languages is crucial for maintaining cultural diversity. The Chakma language, spoken by the Chakma people, faces significant challenges in written script acquisition due to the lack of accessible resources, teaching tools, and learning materials. ChangmaBhach is an ML-powered mobile application designed to address these challenges by providing a comprehensive and interactive platform for learning the Chakma language and script.

Built using Flutter and integrated with TensorFlow Lite, the app enables users to practice Chakma characters by drawing them directly on a canvas. An offline handwriting recognition system validates these inputs, making learning accessible even in connectivity-limited areas.

Features

  • Interactive handwriting practice with real-time recognition and feedback
  • Structured lessons for learning Chakma vowels, consonants, and diacritics
  • Comprehensive dictionary with Chakma-Bangla translations
  • Flashcards and alphabet table for supplemental learning
  • Offline functionality for learning without internet connection
  • Progress tracking and performance feedback
  • Support for both English and Bangla interfaces

Technical Details

Frontend Flutter
ML Framework TensorFlow Lite
Model Type CNN (Convolutional Neural Network)
Dataset 1,280 handwriting samples
Model Accuracy 99% on test data
Platforms Android, iOS

Development Process

Dataset Preparation

Created a custom dataset of 1,280 handwriting samples for Chakma characters, spanning vowels, consonants, and diacritics. Applied preprocessing techniques including standardization and augmentation to improve model robustness.

Model Development

Trained a convolutional neural network (CNN) for handwriting recognition using TensorFlow. Converted the model to TensorFlow Lite for integration with Flutter, optimizing for offline use through model quantization.

App Implementation

Built the app using Flutter for cross-platform compatibility. Implemented features including handwriting practice with real-time feedback, structured lessons, flashcards, and a Chakma-Bangla dictionary, all optimized for offline use.
