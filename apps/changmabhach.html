About this app

ChangmaBhach is a comprehensive Chakma language learning application designed to preserve and promote the Chakma language. The app offers an intuitive interface for users of all ages to learn, practice, and master the Chakma language.

Features

  • Interactive lessons for beginners to advanced learners
  • Chakma alphabet learning with proper pronunciation
  • Vocabulary builder with common phrases and words
  • Daily practice exercises and quizzes
  • Cultural context and historical information
  • Offline mode for learning without internet

Development Process

Research & Design

Extensive research on Chakma language structure and pedagogy. Collaborated with language experts to ensure accuracy and cultural sensitivity.

Development

Built with React Native for cross-platform compatibility. Implemented offline functionality and optimized for low-end devices common in rural areas.

Testing & Feedback

Conducted extensive user testing with both native speakers and language learners. Incorporated feedback to improve user experience and learning outcomes.
