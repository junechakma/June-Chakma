About this app

The Social Meet is an innovative social messaging platform that revolutionizes how people connect online. Unlike traditional messaging apps, The Social Meet uses personality assessments to create meaningful connections between users with compatible personalities.

Users begin by taking a comprehensive personality test, which analyzes their traits, preferences, and communication styles. Based on the results, our intelligent system assigns users to groups where they can interact with like-minded individuals, fostering genuine connections and engaging conversations.

Key Features

  • Comprehensive personality assessment test
  • Smart group assignment based on personality compatibility
  • Real-time messaging with Firebase Stream
  • Intuitive and modern user interface
  • Admin dashboard for group management and analytics
  • Secure authentication and data storage
  • Cross-platform support (iOS & Android)
  • User profile customization

Technical Stack

Development Process

User Research & Design

Conducted extensive research on personality types and social interaction patterns. Designed an intuitive UI/UX that encourages meaningful connections while maintaining user privacy and comfort.

Mobile & Web Development

Built the mobile application using Flutter for cross-platform compatibility. Developed a comprehensive admin dashboard using React and Tailwind CSS for group management and user analytics.

Firebase Integration

Integrated Firebase services including Firestore for data storage, Firebase Stream for real-time messaging, and Firebase Authentication for secure user management. Optimized database queries for scalability.
